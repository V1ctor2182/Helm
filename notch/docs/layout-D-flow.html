交互机制总结(也是要写进 Swift 的状态机):
• hover 进入 → 浏览态(媒体主角),移开 220ms 收起;
• 浏览态里 点 ✎ / 允许·拒绝 / 提问选项 → 进入交互态:面板锁定(橙边、忽略 hover 收起)、获键盘焦点、被点功能升主角、媒体缩条;
• 权限/提问由 agent 事件自动触发,无需 hover 就抢 C 位(灵动岛式打扰);
• 发送成功 / 按 Esc / 点面板外 → 解锁,回浏览态或折叠。
优先级排序:① agent 等权限 > ② agent 提问 > ③ 你正在速记 > ④ 媒体浏览。同时多件事时,高优先级当主角,低的退条/收角标。
