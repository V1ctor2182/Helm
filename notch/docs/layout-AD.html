融合规则(写进 Swift):
• hover → 展开 2×2(四格全在),移开 220ms 收起;
• 媒体 / 日历格只读;速记格点一下 → 面板锁定(橙边、忽略 hover 收起、键盘获焦),发送/Esc/点外解锁;
• agent 格:平时列本机 session;有 agent 等权限/提问 → 该格顶部弹卡 + 折叠态右槽橙呼吸提醒,可直接允许/拒绝/选项;
• 网格始终 2×2 不变形,只是格内内容随状态变活。
